## Why these changes
The Stitch output had a strong visual direction, especially Dashboard and Tailor Workspace. The package now filters the best references and adds the missing product-critical screens using the same crimson WiseResume style and blue WiseHire separation.

## Usage rule
Use this package as visual reference only. Do not copy generated HTML directly into the production app. Apply the theme through existing Tailwind tokens, shadcn/ui components, and shared layouts.
