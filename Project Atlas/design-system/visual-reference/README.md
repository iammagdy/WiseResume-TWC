# WiseResume Stitch Polished Design Pack

This package filters the strongest Stitch-generated screens and adds the missing WiseResume / WiseHire screens in the same visual style.

## Start here
Open `index.html` in a browser.

## Important
This is a visual reference package, not production app code. Use it to guide Codex, Claude, Stitch, or Replit while applying the real design system through the existing app tokens and shared components.

## Included
- `screens/` — polished HTML screen previews
- `original-selected/` — the best original Stitch outputs kept for reference
- `docs/` — relevant design system docs
- `design.md` — AI-agent design brief
- `SCREEN_SELECTION.md` — what was kept and what was rebuilt
- `CODEX_APPLY_PROMPT.md` — prompt for safe implementation


## Fixed version note

This version embeds the CSS inside every HTML screen. Open `index.html` after extracting the zip. If you open a single screen file directly, it will still appear branded because the styling is self-contained.
