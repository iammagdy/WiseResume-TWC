# Arabic Terminology Gate

Status: approved terminology baseline from the Arabic platform plan. Native Gulf-Arabic review is still required before broad production rollout.

## Writing Rules

- Use professional Modern Standard Arabic suitable for Gulf audiences.
- Preserve the English spelling of WiseResume and WiseHire.
- Never translate user-entered CV content when the interface or document locale changes.
- Use Arabic-Indic digits for marketing counters and ordinary interface statistics.
- Use Western digits for CV dates, phone numbers, email addresses, URLs, identifiers, scores, and ATS-sensitive content.
- Use Arabic month names with Western digits in Arabic CVs.
- Use Arabic punctuation in prose and isolate embedded Latin text bidirectionally.
- Do not use uppercase transforms or excessive letter spacing for Arabic.
