## The Atlas is now the one place to trust

The repo used to contain several different documentation folders and Markdown files. Some of them were useful historically, but some still described old systems like Kinde and Supabase as if they were current. That could confuse you and future AI agents.

The cleanup makes `Project Atlas/` the only trusted documentation home. Useful rules from the old governance docs were kept, rewritten for the current Appwrite-native app, and moved into the Atlas. The stale outside docs were removed so future work starts from one clear source of truth.
