# Session Log — 2026-05-13 (DevKit Infrastructure Remediation)

## USER Objective
Resolve systemic failures in the Developer Kit by aligning Appwrite infrastructure, fixing authentication, and repairing broken admin panels/smoke tests.

## Root Causes Identified
1. **Permission Denials:** `visitor_events` lacked `create` permissions for `users` and `guests`, blocking analytics ingestion.
2. **Infrastructure Drift:** 5 collections required for the Portfolio Usernames panel were missing, causing runtime crashes.
3. **Ghost Functions:** DevKit Smoke Runner attempted to invoke non-deployed functions (e.g., `me`, `ai-test`, `manage-api-keys`), causing 14+ false failures.
4. **Incorrect Targets:** `AuditLogPanel` and `EmailManagementPanel` targeted a non-existent `admin-audit-logs` function instead of the `admin_audit_logs` collection.
5. **Secret Mismatch:** `DEVKIT_PASSWORD` was missing from `admin-onboarding-funnel`.
