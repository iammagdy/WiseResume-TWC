# Skill: QA & Test Suite Validation

**Skill ID:** `qa-validation`
**Location:** `Project Atlas/skills/qa-validation.md`

---

## Testing Guidelines

* **Executable Test Code**: Runnable tests live in root `tests/` (`tests/e2e/specs/`, `tests/hubs/`).
* **Generated Test Outputs**: Machine test outputs stay in root `reports/`.
* **Vitest Suite**: Run `npm run test` for unit tests.
* **Playwright E2E**: Run `npx playwright test` for end-to-end browser tests.
* **QA Maps**: Consult `Project Atlas/qa/test-suite-map.md` and `Project Atlas/qa/test-output-locations.md`.
