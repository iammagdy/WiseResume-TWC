## Visual Design Standards

* **Design System**: Use Tailwind CSS v4, Radix UI primitives, and shadcn/ui components.
* **Target Visual Specs**: Consult `Project Atlas/design-system/production/` and `visual-reference/`.
* **Color Palette**: WiseResume primary brand color `#9E1B22`. Dark mode glassmorphic styling for DevKit and premium controls.
* **Responsive Layouts**: Design for desktop (1440px) and mobile (390x844px).
* **Animations**: Micro-animations via Framer Motion or subtle CSS transitions.
