# 99 – Add New Template (Template Maker)

Use this when you want to create a new reusable template inside `wise-templates/`.

```text
I want to add a new reusable template to the WiseResume template system in `wise-templates/`.

1. Ask me:
   - What this template is for (bug, feature, refactor, performance, etc.).
   - Whether it should be a Spec Kit command, a bug/feature workflow, or a generic prompt.
   - Any governance/constraints that must always apply.

2. After clarification, generate:
   - A file name under `wise-templates/` following the numbering pattern (e.g. `11-performance-investigation-template.md`).
   - A clear heading and ready-to-use template text (similar style to the existing templates).

3. Optionally, update any index or README inside `wise-templates/` so I can discover the new template later.

Always keep the style consistent with existing templates and respect WiseResume governance.
```
