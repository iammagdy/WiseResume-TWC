```text
You are an AI agent working inside the WiseResume repository.

## Context

- WiseResume has a Settings or "What’s New" section showing recent changes.
- A new version has been released: <NEW VERSION>.

## Task

1. Find where changelog content is stored (components, JSON, markdown, etc.).
2. Add a new entry for version <NEW VERSION>:
   - Short title.
   - Bullet list of key changes.
   - Date.

3. Ensure:
   - No legacy branding or Supabase Auth references.
   - Mobile-first layout and dark theme are preserved.
   - Design tokens and typography rules are respected.

4. Keep older entries intact unless clearly wrong.

5. Summarize:
   - New UI text/content added.
   - Files/components touched.
```
