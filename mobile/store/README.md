# Store listings

Drop final store assets here once the brand team approves them. EAS
Submit reads from the App Store Connect / Play Console projects directly,
so this directory is purely a source-of-truth checkpoint for review.

## Required assets

### Apple App Store

- `ios/icon-1024.png` (1024×1024, no alpha)
- `ios/screenshots/6.7-inch/*.png` (1290×2796) — minimum 3
- `ios/screenshots/6.5-inch/*.png` (1284×2778)
- `ios/screenshots/5.5-inch/*.png` (1242×2208)
- `ios/preview.mp4` (optional 15-30s app preview)
- `ios/description.txt` (≤ 4000 chars)
- `ios/keywords.txt` (≤ 100 chars, comma-separated)
- `ios/promotional-text.txt` (≤ 170 chars)
- `ios/privacy-nutrition-label.md` — App Privacy questionnaire answers

### Google Play

- `android/icon-512.png` (512×512)
- `android/feature-graphic.png` (1024×500)
- `android/screenshots/phone/*.png` (1080×1920) — minimum 4
- `android/screenshots/7-inch/*.png`
- `android/screenshots/10-inch/*.png`
- `android/short-description.txt` (≤ 80 chars)
- `android/full-description.txt` (≤ 4000 chars)
- `android/data-safety.md` — Data Safety questionnaire answers

## Copy guidelines

* Lead with the outcome: "Land more interviews, faster."
* Mention Pro features but never claim outcomes you can't substantiate.
* Avoid generic AI superlatives — describe specific capabilities
  (resume scoring, ATS rewrite, voice-graded interview practice).
* Include the unique mobile-only benefits: voice interview practice,
  push reminders for follow-ups, biometric lock, offline reading.
