## 📝 How to Use This File

1. **Before designing each screen**, find 2-3 competitors above that have a similar screen
2. **Screenshot their approach** using Mobbin or manual screenshots
3. **Feed screenshots to Google Stitch** alongside your prompt from `STITCH_PROMPTS_WEB.md`
4. **Adapt, don't copy** — WiseResume has its own dark space theme, glassmorphism, and brand identity
5. **Focus on interaction patterns** more than visual style — how does their stepper work? How do they show loading states?
